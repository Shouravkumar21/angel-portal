# Ascencia Business School Malta – Static Frontend

A static React frontend inspired by [Ascencia Business School Malta](https://www.ascencia-business-school.mt/), with a new UI theme aligned to the existing logo and brand. UI only (no backend or API).

## Pages

- **Home** (`/`) – Hero, courses, about preview, Why Floriana, welcome message, campuses, CTA
- **About Us** (`/ascencia-malta`) – Mission, pedagogy, values, tutors
- **Contact** (`/contact`) – Contact info and contact form (UI only, no submission)

## Tech Stack

- React (functional components)
- React Router
- Plain CSS (no Bootstrap, Tailwind, or MUI)
- Vite

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/`.

## Responsive

Layouts are responsive for desktop, tablet, and mobile (breakpoints at 1024px and 768px).
